Gráfico 1

O primeiro gráfico mostra a relação entre os gêneros e os oscars, ou seja, a quantidade de oscars ganhos pelos os filmes, da nossa lista de filmes, divido por gênero.

Para conseguir setar as cores no mesmo ou próximo do gradiente do gerado pelo tableau, fui buscar ajuda no site: ColorBrewer para pegar os valores rgb's das cores das colunas, logo em seguida tive que setar as opções de cores através do Highcharts.setOptions({ colors: ['#023858', '#034e7b', '#0570b0', '#0570b0', '#74a9cf', '#74a9cf', '#d0d1e6', '#d0d1e6'] });, desta forma as cores foram setados de acordo com seu valor.

Uma das difilcudades encontradas foi setar os valores de cada barra logo acima dela, apesar de adicionado no código fonte, não consegui fazer com que os valores ficassem visíveis. Outra mudança foi na forma de mostrar os nomes de cada barra, foi incluida uma caixa dentro do gráfico, desta forma, economizando espaço na página.
